power saving mode you can set up to have a black and white color display, restrict background data, and restrict performance. I generally have this turned on anytime I'm not actively using the phone.

Ultra power saving mode shuts down uneeded processes and puts it into a special operating mode that really conserves battery. On a full charge I've had my phone on standby for as long as 10 days. using it to text and make calls will reduce the time but it can be usefull for long hauls.

Additionally there's some other things you can do to save battery power such as disabling wifi when you're not actively using the phone. this will shut off the wifi antenae and save juice. disable NFC and bluetooth wen not being used or if you don't use them at all. etc.
